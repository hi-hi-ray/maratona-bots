# BotTranslatorUnicorn

Esse é o meu projeto do Maratona Bots.

## Sobre

O Bot Translator Unicorn, é bot voltado para traduções para português, e também consegue descrever uma foto a partir do url da imagem. 

### O que está utilizando:

- LUIS
- TRANSLATOR TEXT API
- VISION API
- PLATAFORMA AZURE
